The Vidya-Platformer package is an abandoned project of mine that was meant to help significantly reduce development time on creating two-dimensional platformer style web-based video games.

The project was put to the wayside due to work related circumstances and other more lucrative projects. It will be resumed at a point when I have more free time.

This package is in an extremely alpha state and may not function as intended on all browsers.

This package was intended to be ran on a desktop computer and requires a keyboard. It will not function on mobile devices.

Foreword

At first glance, this demonstration is admittedly underwhelming and unimpressive.

However, the reason why I feel this is impressive, and why I included it, is that when invoking Semantic UI dropdown method on an element, it essentially undoes any ties to Angular on the element.

What I've done is allowed the element to have the semantic UI dropdown invoked on it, and still have the dropdown's value be bound to the scope.

Demonstration Markup

<div class="field">
	<label>Select Single Icon</label>
	<dropdown ng-model="dropValue" placeholder="Please select a single icon">
		<choice ng-repeat="item in drops" choice-value="item.icon" display="item.icon"></choice>
	</dropdown>
	<div class="ui info message">{{dropValue}} <i class="{{dropValue}} icon"></i></div>
</div>

<div class="field">
	<label>Select Multiple Icons</label>
	<dropdown ng-model="dropValueMulti" multiple="true">
		<choice ng-repeat="item in drops" choice-value="item.icon" display="item.icon"></choice>
	</dropdown>
	<div class="ui info message">{{dropValueMulti}} <div ng-repeat="item in dropValueMulti"><i class="{{item}} icon"></i></div></div>
</div>
